# /vault

Save, browse, and install community components.

## Subcommands

### /vault browse
Browse all components in the community Vault.

### /vault search [query]
Search by keyword, tag, framework, or design system.

### /vault save
Package and save the current component (poke-selected or specified) to the Vault.

### /vault install [name]
Download and install a Vault component into your project.

### /vault my-components
List your saved components.

### /vault share [name]
Generate a shareable link for a component.

### /vault competition create [name] --deadline [date] --prize [amount]
Create a new component competition.

### /vault competition submit [component-name]
Submit a component to an active competition.

### /vault competition leaderboard [competition-id]
View the competition leaderboard.
